# **App Name**: JamaStore

## Core Features:

- Multi-Language Lookbook Display: Browse curated 'Looks' (outfits) with full localization for Uzbek, Russian, and English, adapting UI and content for each language.
- AI Smart Size Advisor: Recommend optimal clothing sizes (S-XXL) based on user-provided height, weight, gender, and desired fit, using Genkit-powered logic.
- Admin Look Management: Secure admin dashboard to create, edit, and upload high-resolution images for new 'Looks' in the clothing catalog.
- AI Admin Content Generator: Utilize an AI tool to generate professional, SEO-friendly multi-language product descriptions for admin-managed 'Looks' from a few keywords.
- Order Status Management: Admins can efficiently track and update customer order statuses (e.g., New, Confirmed, Shipped) through the dashboard.
- Real-time Telegram Notifications: Send automated order status updates to customers and new order alerts to admins via Telegram Bot API, with AI composing messages.
- User Account Creation & Storage: Allow users to create secure accounts using email and link their Telegram username, storing details in Firebase.

## Style Guidelines:

- Primary background color: Dark Grey (#1A1A1A) for a futuristic and sophisticated aesthetic.
- Accent and interactive elements (buttons, highlights): Vibrant Yellow (#FFC107) to provide energy and clear calls to action.
- Secondary text and subtle UI elements: Light Grey (#CCCCCC) for contrast against the dark background.
- Body and headline font: 'Inter' sans-serif for modern legibility, complementing the futuristic design.
- Minimalist and sharp icons that align with a modern, futuristic aesthetic, integrated seamlessly with Shadcn UI.
- Mobile-first responsive design with a futuristic and clean layout, featuring spacious elements, generous rounded corners (2xl), and innovative UI patterns.
- Extensive use of subtle yet engaging fluid transitions, micro-interactions, and glassmorphism (backdrop-blur) for a dynamic and futuristic user experience.
